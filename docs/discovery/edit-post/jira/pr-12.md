## Description

Creates a new, unpublished post when the "New Post" button is clicked, and redirects to the edit page of the newly created post.

**`createPost` update:** Instead of redirecting to the post page,
`createPost` creates a post and redirects to `ROUTES.editPost(post.id)`. The
draft is created with a timestamped placeholder title (e.g. `"Untitled —
2026-05-02 10:30:45"`).

**`PostPageAdminMenuContent` after this PR:**

| Action | Implementation |
|--------|---------------|
| New Post | `<form action={createPost}>` |

## Acceptance Criteria

- [ ] Clicking "New Post" creates a draft and redirects to `/posts/[id]/edit`
- [ ] `pnpm validate` passes

## Technical Notes

- The timestamped placeholder title (e.g. `"Untitled — 2026-05-02 10:30:45"`) is subject to the `@unique` constraint on `Post.title`. Two drafts created within the same second would collide — acceptable at this scale. See `decisions.md` → D23 and D8.
