## Description

Update the styles to match the Figma designs for the title input.

**`TitleInput`:**
- Styled to be invisible: no border, no background — visually integrated with the page content.
- On change, updates `title` state in `EditPostForm`, which triggers the autosave debounce.
- On unique-constraint error from autosave, renders an inline error above the input (→ D37, D43).


**Inline error states (deferred from PR 7):**
- `state.threwUniqueConstraintError` renders a hard-coded error message above `TitleInput`.
- `state.dtoError?.fieldErrors?.title` renders inline errors above the input.

## Acceptance Criteria

- [ ] `TitleInput` has no visible border or background
- [ ] `TitleInput` changes trigger autosave via `EditPostForm` state
- [ ] `state.threwUniqueConstraintError` renders a hard-coded error message above `TitleInput`
